# MtF-Certificate

MtF 获取医疗资源相关的医疗证明模板

鉴于目前跨性别相关医疗环境日渐恶化，越来越多的跨儿难以获取 HRT 所需的医疗证明
本项目希望为难以获取相关资源的跨儿提供模板，增加获得 HRT 处方的机会
